[Matt Rooney](https://github.com/mattrooney)

##### About
This is a Javascript game based on the retro game, Snake! To run this game locally, simply:

Clone this repo:

`git clone git@github.com:adamki/snake-time.git`

Install dependencies

`npm install`

Start the Server:

'npm start'

Visit in Browser: 

`http://localhost:8080`
